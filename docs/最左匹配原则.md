- [[索引]]基础
-
- 使用[[联合索引]]时，存在**最左匹配原则**，也就是按照最左优先的方式进行索引的匹配。
- 有[[查询优化器]]，所以 a 字段在 where 子句的顺序会调整
-
- 最左匹配原则失效 #when #card
	- 范围查询无等号 会停止匹配
		- 在遇到范围查询（如 >、<）的时候，就会停止匹配，也就是范围查询的字段可以用到联合索引，但是在范围查询字段后面的字段无法用到联合索引。
		- 注意，对于 >=、<=、BETWEEN、like 前缀匹配的范围查询，并不会停止匹配。
